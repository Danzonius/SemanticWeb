Explanation

A user can write a sparql query in the textarea above which, if written correctly, can retrieve data from a stardog triple store. There are two buttons which the user can click on. One button retrieves data with a reasoner turned on and the other with a reasoner turned off. The retrieved data with the reasoner turned off, is data which gives the user more information about the structure and the semantics of the classes and objects used in the ontology. With the reasoner turned on, the data gives more information about the relations between objects, classes and their properties.
